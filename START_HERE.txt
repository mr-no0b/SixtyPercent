Step 1: Install MySQL/MariaDB (if not installed)
─────────────────────────────────────────────────
    sudo pacman -S mariadb
    sudo systemctl start mariadb

Step 2: Run the setup script
─────────────────────────────
    ./setup.sh

Step 3: Start the application
──────────────────────────────
    ./start.sh

Done! Open browser → http://localhost:5000

Problem: MySQL not starting
Solution: sudo systemctl start mariadb

Problem: Import errors
Solution: Make sure venv is activated (source venv/bin/activate)

Problem: Database connection failed
Solution: Edit .env file with correct MySQL password

Problem: Port 5000 busy
Solution: Edit app.py, change port to 5001

For more help, check README.md section "Troubleshooting"
