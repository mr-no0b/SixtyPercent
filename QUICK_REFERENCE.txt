📊 UNDERSTANDING THE CALCULATIONS
═══════════════════════════════════════════════════════════════════════

Example: 40 Total Classes, 60% Required

SCENARIO 1: Safe Zone
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Present: 20, Absent: 5 (25 held, 15 remaining)
Current %: 20/25 = 80% ✅

Required Present: 40 × 60% = 24 classes
Classes Needed: 24 - 20 = 4 more
Classes Can Miss: 15 - 4 = 11 classes

STATUS: ✅ SAFE - Can miss 11 more

SCENARIO 2: Danger Zone ⚠️
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Present: 16, Absent: 16 (32 held, 8 remaining)
Current %: 16/32 = 50% ❌

Required Present: 40 × 60% = 24 classes
Classes Needed: 24 - 16 = 8 more
Remaining: 8
Classes Can Miss: 0

STATUS: ⚠️ DANGER ZONE - Must attend ALL 8 remaining!

SCENARIO 3: Dead Zone ☠️
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Present: 15, Absent: 20 (35 held, 5 remaining)
Current %: 15/35 = 42.86% ❌

Required Present: 40 × 60% = 24 classes
Maximum Possible: 15 + 5 = 20 (< 24)

STATUS: ☠️ DEAD ZONE - Impossible to reach 60%

═══════════════════════════════════════════════════════════════════════
🎨 VISUAL INDICATORS
═══════════════════════════════════════════════════════════════════════

COLORS:
✅ Green Numbers = Good (Present count, Can Miss > 0, Current % >= Target)
❌ Red Numbers = Bad (Absent count, Can Miss = 0, Current % < Target)
🔵 Blue Numbers = Neutral (Remaining classes)
🟡 Yellow/Orange = Warning (Classes Needed > 0)

ALERTS:
⚠️ DANGER ZONE = Red background, pulsing, can't miss any more
☠️ DEAD ZONE = Black background, pulsing, impossible to recover

BUTTONS:
✏️ Edit = Blue-gray (modify details)
🗑️ Delete = Red (remove item)
➕ Add = Purple (create new)

🚀 QUICK ACTIONS GUIDE
═══════════════════════════════════════════════════════════════════════

CREATE COURSE WITH REQUIRED TOTAL:

EDIT TEACHER:
─────────────
My Teachers → Find Teacher → ✏️ Edit
Update: Name, Email, Department
💾 Save Changes

MARK ATTENDANCE:
────────────────
Course Detail → Mark Attendance
📊 See: Total (40) | Marked (25) | Remaining (15)
Select Date, Status, Notes → 💾 Save

═══════════════════════════════════════════════════════════════════════
📚 FORMULAS REFERENCE
═══════════════════════════════════════════════════════════════════════

Required Present = (Required % × Total Classes) / 100
Classes Needed = Required Present - Current Present (min: 0)
Max Absences = Total Classes - Required Present
Classes Can Miss = Max Absences - Current Absences (min: 0)
Remaining = Total Classes - Classes Held

Danger Zone = (Remaining > 0) AND (Remaining == Classes Needed)
Dead Zone = (Present + Remaining) < Required Present
